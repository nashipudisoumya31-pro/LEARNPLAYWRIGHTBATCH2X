# LearnPlaywright Batch 2x

A structured learning repository covering JavaScript fundamentals as a foundation for Playwright test automation.

|------|-------|

| `02_JS.js` | Core JS concepts |
| `03_JS_Verify_Setup.js` | Verifying the development setup |
| `04_HotCode.js` | Hot code reloading |

### Chapter 02 - JavaScript Concepts
| File | Topic |
|------|-------|

### Chapter 03 - Identifier Literals

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [Visual Studio Code](https://code.visualstudio.com/)
- [Playwright](https://playwright.dev/)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/nashipudisoumya31-pro/LEARNPLAYWRIGHTBATCH2X.git

# Navigate into the project
cd LEARNPLAYWRIGHTBATCH2X

# Install dependencies
npm install
```

## Running Scripts

```bash
node chapter_01_Basics/01_Basics.js
```
